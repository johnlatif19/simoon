## 📖 عن المشروع

منصة سياحية متكاملة لإدارة الحجوزات والرحلات السياحية في مصر. توفر المنصة تجربة سلسة للعملاء لحجز الرحلات، مع لوحة تحكم متكاملة للمديرين لإدارة المحتوى والحجوزات والتقييمات.

### ✨ المميزات الرئيسية

- 🏝️ **إدارة الرحلات** - CRUD كامل للرحلات السياحية مع صور وبرامج مفصلة
- 📅 **نظام حجز متقدم** - حساب الأسعار تلقائياً حسب الجنسية وعدد الأفراد
- 💳 **بوابة دفع** - دعم أورنج كاش مع نظام تحويل مرجعي
- ⭐ **نظام تقييمات** - آراء العملاء مع تصفية وعرض متوسط التقييمات
- 📊 **لوحة تحكم احترافية** - إحصائيات، إدارة حجوزات، رسائل العملاء
- 📧 **إشعارات بريدية** - تأكيد الحجز وإشعارات المسؤول تلقائياً
- 🌙 **الوضع الليلي** - دعم كامل للوضع المظلم
- 🌍 **دعم لغتين** - العربية والإنجليزية
- 📱 **تصميم متجاوب** - يعمل على جميع الأجهزة

## 🛠️ التقنيات المستخدمة

### Backend
- **Node.js** - بيئة التشغيل
- **Express.js** - إطار العمل
- **Firebase Admin SDK** - قاعدة البيانات
- **JSON Web Tokens** - المصادقة
- **Nodemailer** - إرسال الإيميلات

### Frontend
- **HTML5/CSS3** - الهيكل والتصميم
- **JavaScript (ES6+)** - التفاعلات
- **Font Awesome** - الأيقونات
- **Google Fonts** - الخطوط (Cairo)

### الأدوات والتطوير
- **Vercel** - الاستضافة والنشر
- **Git** - التحكم في الإصدارات
- **Nodemon** - التطوير المحلي
